# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Flask that simulates ATS (Applicant Tracking System) screening. It evaluates resumes based on content quality, keyword optimization, formatting, and spelling accuracy, and provides a detailed score along with actionable improvement suggestions.

---

## 📌 Features

- ✅ ATS Score Calculation (0–100)
- 🔍 Keyword Matching & Optimization
- 📝 Content Quality Analysis
- ⚠️ Error Detection (Spelling & Formatting)
- 📊 Section-wise Feedback (Skills, Experience, etc.)
- 💡 Smart Suggestions to Improve Resume
- 🌐 Clean and Responsive UI
